# QuickCart – AI-Powered E-Commerce Platform

## Overview

QuickCart is a full-stack AI-powered e-commerce web application designed to provide a seamless online shopping experience. Users can browse products, view product details, add items to their cart, place orders, submit reviews, and receive product recommendations through an AI shopping assistant chatbot.

The platform also includes an admin panel for managing products and categories.

## Workflow

1. User opens QuickCart.
2. Products are fetched from MongoDB.
3. User browses products.
4. Product details are displayed.
5. User adds products to cart.
6. Cart data is stored and managed.
7. User places an order.
8. Order information is saved.
9. Reviews are submitted for specific products.
10. AI chatbot provides shopping assistance.
